== Description ==

This plugin adds the required javascript for Nirror.

For more information visit:

[Nirror](http://www.nirror.com)

== Installation ==

1. Upload `nirror` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set the `Site ID` from Nirror in the settings (Admin > Settings > Nirror)
